# WordPress-Login-Flow-Android

A pluggable WordPress login flow for Android.

## Usage

```groovy
repositories {
    maven { url "https://a8c-libs.s3.amazonaws.com/android" }
}
dependencies {
    implementation ("org.wordpress:login:$version")
}
```
